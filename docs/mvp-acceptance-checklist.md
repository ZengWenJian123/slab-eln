# MVP 验收清单

## 核心流程

- [ ] 成功流程：成分 -> 熔炼 -> 拉丝 -> 样品 -> 后处理 -> 测试 -> 图片关联
- [ ] 失败流程：任一工序状态设为 `FAILED` 时必须填写失败原因
- [ ] 样品详情可聚合展示后处理、测试、图片记录

## 权限

- [ ] ADMIN 可管理用户与字典
- [ ] OPERATOR 可创建/编辑实验数据，不可管理用户
- [ ] VIEWER 只读访问

## 规则

- [ ] 成分总和必须为 100%
- [ ] 熔炼损耗率由后端重算
- [ ] 玻璃层厚度由后端重算
- [ ] 失败原因前后端双校验

## 文件与图片

- [ ] CSV/TXT 可上传并写入附件元数据
- [ ] 图片可上传并写入图片元数据
- [ ] 图片可按样品/分类/阶段筛选

## 部署

- [ ] `.env` 配置可用
- [ ] `npm run build` 成功
- [ ] `npm run test` 成功
